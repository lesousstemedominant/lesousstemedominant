# Le Multi-Greg

Greg est le martyr de Jumanjaaj, néanmoins il existe une infinité de Greg, 1 par réalité. Nous pouvons par exemple citer Györn le Greg hongrois ; Finkel le Greg Parisien noyé devant le jardin Villemin dans le canal qui porte maintenant son nom, mais aussi L’Ondine ou Greg Celte
